# 📝 Notely REST API

A REST API for managing notes, built to demonstrate backend development concepts such as CRUD operations, API endpoints, and database interaction.

---

## 🚀 Features

* Create notes
* Retrieve notes
* Update existing notes
* Delete notes
* JSON-based API responses

---

## 🛠 Technologies

* C#
* .NET
* REST API
* SQLite
* Postman (API testing)

---

## 📂 Project Structure

```
NotelyRestApi
│
├── Controllers
├── Models
├── Services
├── Database
└── Program.cs
```

---

## 📡 API Endpoints

| Method | Endpoint    | Description         |
| ------ | ----------- | ------------------- |
| GET    | /notes      | Get all notes       |
| GET    | /notes/{id} | Get a specific note |
| POST   | /notes      | Create a new note   |
| PUT    | /notes/{id} | Update a note       |
| DELETE | /notes/{id} | Delete a note       |

---

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/brunooc11/notely-rest-api.git
```

2. Navigate to the project folder

```
cd notely-rest-api
```

3. Run the API

```
dotnet run
```

4. Test the endpoints using **Postman** or any API client.
